SKrulll Logo Conversion Tool

This page displays the SKrulll logo in SVG format. You can use this to create a PNG version for the README.

Instructions to Save as PNG:

  1. Take a screenshot of the logo above (make sure to capture just the logo)
  2. Open the screenshot in an image editor
  3. Crop to include only the logo
  4. Save as logo.png in the static/img/ directory

Alternative Method (Using Browser):

  1. Right-click on the logo above
  2. Select "Save Image As..." (or similar option in your browser)
  3. Save it as logo.png in the static/img/ directory
  4. Note: Some browsers may save it as SVG instead of PNG

Using Online Converters:

You can also use online SVG to PNG converters:

  1. Copy the SVG code below
  2. Paste it into an online converter like svgtopng.com
  3. Download the PNG and save it as logo.png in the static/img/ directory
